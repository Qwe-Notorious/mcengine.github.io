# MCEngine

**MCEngine** — это мощный Fabric-мод, который интегрирует в Minecraft полноценный JavaScript-движок на базе GraalVM. 
Он позволяет создавать сложную игровую логику, кастомных мобов с уникальным AI и динамические события без необходимости писать и компилировать Java-код.

---

## Возможности

<div class="cards" markdown>

-   **Продвинутый AI и Хуки**

    Перехватывай тики, атаки, получение урона и смерть мобов. Управляй их поведением через JS-коллбэки или полностью отключай ванильный AI.
    
    ---

-   **Кастомные мобы и Боссы**

    Создавай уникальных существ с заданным здоровьем, именами, моделями и скриптами поведения прямо из чата или стартового скрипта.

    ---

-   **Динамическая замена моделей**

    Подменяй ванильные текстуры и модели мобов на кастомные (глобально или для конкретного экземпляра) через систему Resource Pack overrides.
    
    ---

-   **Встроенный редактор кода**

    Редактируй скрипты прямо внутри игры. Встроенный редактор поддерживает дерево файлов, вкладки и подсветку синтаксиса.

    ---

-   **Управление игроками**

    Выдавай предметы, телепортируй, меняй режимы игры, устанавливай точки спавна и отправляй сообщения в чат или Action Bar.

    ---

-   **Высокая производительность**

    Благодаря GraalVM Polyglot, JavaScript выполняется непосредственно в JVM, обеспечивая быстрый доступ к API Minecraft и безопасность.

</div>

## Быстрый старт
- Установка: Скачай последнюю версию мода и помести .jar файл в папку mods твоего клиента/сервера.
- Создание проекта: Запусти игру, открой главное меню (или меню паузы) и нажми кнопку MCEngine. Создай новый проект.
- Написание кода: Открой встроенный редактор кода, создай файл main.js и начни писать скрипты.
- Перезагрузка: Используй команду /mcengine reload, чтобы применить изменения без перезапуска игры.

!!! tip "Совет"
    Ты можешь выполнять сырой JavaScript код прямо из чата с помощью команды:
    /mcengine run console.log("Hello MCEngine!");

## Пример кода

Вот как просто можно изменить поведение моба:

```javascript

// Модификация AI мобов
MCEngineServer.modifyAI("minecraft:zombie", {
    onTick: (mob) => {
        var player = mob.findNearestPlayer(15);
        if (player != null) {
            var dx = mob.getX() - player.getX();
            var dz = mob.getZ() - player.getZ();
            var dist = Math.sqrt(dx * dx + dz * dz);
            if (dist > 0) {
                mob.moveTo(mob.getX() + (dx / dist) * 5, mob.getY(), mob.getZ() + (dz / dist) * 5, 1.2);
            }
        }
    },
    
    onDeath: (mob) => {
        mob.spawnParticle("minecraft:large_smoke", 1, 10);
    }    
    
});
conosle.log("Зомби трус !!!")

```
