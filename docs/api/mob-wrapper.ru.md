# Обертка моба (MobWrapper)

`MobWrapper` — это безопасная обертка над сущностями Minecraft (`Entity`, `LivingEntity`, `MobEntity`), предназначенная для использования в JavaScript. Она адаптирована под API Minecraft 1.21.11 и предоставляет удобный доступ к свойствам и методам мобов без необходимости работать с низкоуровневым Java-кодом.

!!! note "Примечание"
    Экземпляр `MobWrapper` создается автоматически при передаче моба в JS-коллбэки (например, в `onTick`, `onDeath`) или при использовании методов поиска и спавна.

---

## Базовая информация

### getType()
Возвращает непереведенное имя типа сущности.

**Возвращает:** `String` (например, `"Zombie"`).

### getFullType()
Возвращает полный идентификатор типа сущности из реестра.

**Возвращает:** `String` (например, `"minecraft:zombie"`).

### getUuid()
Возвращает уникальный идентификатор сущности.

**Возвращает:** `String` (UUID в строковом формате).

### getName()
Возвращает текущее отображаемое имя сущности.

**Возвращает:** `String`.

### isAlive()
Проверяет, жива ли сущность.

**Возвращает:** `boolean`.

---

## Позиция и движение

### getX(), getY(), getZ()
Возвращают текущие координаты сущности.

**Возвращает:** `Number` (double).

### getYaw(), getPitch()
Возвращают текущий угол поворота (рыскание и тангаж) сущности.

**Возвращает:** `Number` (float).

### setPosition(x, y, z)
Мгновенно перемещает сущность на указанные координаты без анимации.

**Параметры:**

| Параметр | Тип    | Описание           |
| :------- | :----- | :----------------- |
| `x, y, z`| Number | Целевые координаты |

### teleport(x, y, z)
Телепортирует сущность. 

!!! info "Особенности реализации"
    Для игроков (`ServerPlayerEntity`) используется безопасный метод `requestTeleport`. Для остальных сущностей используется метод `teleport` с привязкой к `ServerWorld`, что предотвращает ошибки в Minecraft 1.21+.

**Параметры:**

| Параметр | Тип    | Описание           |
| :------- | :----- | :----------------- |
| `x, y, z`| Number | Целевые координаты |

### moveTo(x, y, z, speed)
Заставляет моба самостоятельно идти к указанным координатам, используя встроенный навигатор Minecraft.

!!! warning "Внимание"
    Работает только для сущностей, наследующих `MobEntity`. Для игроков или других сущностей вернет `false`.

**Параметры:**

| Параметр | Тип    | Описание                                |
| :------- | :----- | :-------------------------------------- |
| `x, y, z`| Number | Целевые координаты                      |
| `speed`  | Number | Скорость движения (обычно от 0.1 до 1.0)|

**Возвращает:** `boolean` (успешно ли начато движение).

### stopMoving()
Немедленно останавливает движение моба.

## Здоровье и урон

### getHealth()
Возвращает текущее здоровье сущности.

**Возвращает:** `Number` (float). Возвращает `0`, если сущность не является `LivingEntity`.

### setHealth(health)
Устанавливает здоровье сущности.

**Параметры:**

| Параметр | Тип    | Описание                |
| :------- | :----- | :---------------------- |
| `health` | Number | Новое значение здоровья |

### getMaxHealth()
Возвращает максимальное здоровье сущности.

**Возвращает:** `Number` (float).

### kill()
Мгновенно убивает сущность.

!!! info "Особенности реализации"
    В Minecraft 1.21+ метод `kill()` требует передачи `ServerWorld`. Обертка делает это автоматически.

---

## Имя и внешность

### setName(name)
Устанавливает кастомное имя сущности (поддерживает цветовые коды Minecraft, например `"§cКрасный"`).

**Параметры:**

| Параметр | Тип    | Описание  |
| :------- | :----- | :-------- |
| `name`   | String | Новое имя |

### setNameVisible(visible)
Делает кастомное имя видимым или скрытым над головой сущности.

**Параметры:**

| Параметр  | Тип     | Описание                     |
| :-------- | :------ | :--------------------------- |
| `visible` | Boolean | `true` для отображения, `false` для скрытия |

### setInvisible(invisible)
Делает сущность невидимой.

**Параметры:**

| Параметр    | Тип     | Описание                |
| :---------- | :------ | :---------------------- |
| `invisible` | Boolean | `true` для невидимости  |

### setSilent(silent)
Отключает все звуки, издаваемые сущностью.

**Параметры:**

| Параметр | Тип     | Описание                 |
| :------- | :------ | :----------------------- |
| `silent` | Boolean | `true` для отключения звуков |

### setNoGravity(noGravity)
Отключает гравитацию для сущности (она будет парить).

**Параметры:**

| Параметр    | Тип     | Описание                  |
| :---------- | :------ | :------------------------ |
| `noGravity` | Boolean | `true` для отключения гравитации |

## Цели и атаки

### setTarget(target)
Заставляет моба выбрать новую цель для атаки.

**Параметры:**

| Параметр | Тип        | Описание               |
| :------- | :--------- | :--------------------- |
| `target` | MobWrapper | Обертка целевой сущности |

### getTarget()
Возвращает текущую цель атаки моба.

**Возвращает:** `MobWrapper` (цель) или `null`, если цели нет.

### attackTarget(target)
Заставляет моба немедленно выполнить анимацию и логику атаки по указанной цели.

!!! info "Особенности реализации"
    В Minecraft 1.21+ метод `tryAttack` требует передачи `ServerWorld`. Обертка делает это автоматически.

**Параметры:**

| Параметр | Тип        | Описание               |
| :------- | :--------- | :--------------------- |
| `target` | MobWrapper | Обертка целевой сущности |

**Возвращает:** `boolean` (успешна ли атака).

---

## Поиск сущностей

### findNearestPlayer(radius)
Ищет ближайшего игрока в заданном радиусе.

**Параметры:**

| Параметр | Тип    | Описание              |
| :------- | :----- | :-------------------- |
| `radius` | Number | Радиус поиска в блоках|

**Возвращает:** `PlayerWrapper` (ближайший игрок) или `null`, если игроков нет в радиусе.

### findNearestMob(entityType, radius)
Ищет ближайшего живого моба указанного типа в заданном радиусе.

**Параметры:**

| Параметр     | Тип    | Описание                                      |
| :----------- | :----- | :-------------------------------------------- |
| `entityType` | String | Тип моба (например, `"minecraft:zombie"`)     |
| `radius`     | Number | Радиус поиска в блоках                        |

**Возвращает:** `MobWrapper` или `null`.

### findAllMobs(entityType, radius)
Находит всех живых мобов указанного типа в заданном радиусе.

**Параметры:**

| Параметр     | Тип    | Описание                               |
| :----------- | :----- | :------------------------------------- |
| `entityType` | String | Тип моба                               |
| `radius`     | Number | Радиус поиска в блоках                 |

**Возвращает:** `Array` (массив объектов `MobWrapper`).

### distanceTo(other)
Вычисляет расстояние до другой сущности.

**Параметры:**

| Параметр | Тип        | Описание         |
| :------- | :--------- | :--------------- |
| `other`  | MobWrapper | Целевая сущность |

**Возвращает:** `Number` (расстояние в блоках). Возвращает `-1`, если `other` равен `null`.

### distanceToPlayer(player)
Вычисляет расстояние до игрока.

**Параметры:**

| Параметр | Тип           | Описание      |
| :------- | :------------ | :------------ |
| `player` | PlayerWrapper | Целевой игрок |

**Возвращает:** `Number` (расстояние в блоках). Возвращает `-1`, если `player` равен `null`.

## Специальные способности

### addEffect(effectId, duration, amplifier)
Накладывает эффект зелья на сущность.

!!! info "Особенности реализации"
    В Minecraft 1.21+ для добавления эффектов требуется `RegistryEntry`. Обертка автоматически преобразует строковый ID в нужный формат.

**Параметры:**

| Параметр    | Тип    | Описание                                           |
| :---------- | :----- | :------------------------------------------------- |
| `effectId`  | String | ID эффекта (например, `"minecraft:speed"`)         |
| `duration`  | Number | Длительность в тиках (20 тиков = 1 секунда)        |
| `amplifier` | Number | Уровень эффекта (0 = уровень I, 1 = уровень II)    |

### setOnFire(seconds)
Поджигает сущность на указанное количество секунд.

!!! info "Особенности реализации"
    В Minecraft 1.21.11 метод принимает значение типа `float`.

**Параметры:**

| Параметр  | Тип    | Описание                  |
| :-------- | :----- | :------------------------ |
| `seconds` | Number | Количество секунд горения |

### playSound(soundId, volume, pitch)
Проигрывает звук на текущей позиции сущности.

**Параметры:**

| Параметр  | Тип    | Описание                                              |
| :-------- | :----- | :---------------------------------------------------- |
| `soundId` | String | ID звука (например, `"minecraft:entity.zombie.ambient"`)|
| `volume`  | Number | Громкость (обычно 1.0)                                |
| `pitch`   | Number | Высота тона (обычно 1.0)                              |

## Комплексный пример: Умный страж

Ниже приведен пример того, как методы `MobWrapper` используются вместе для создания интеллектуального поведения:

```javascript
MCEngineServer.onInit(() => {
    var boss = MCEngineServer.createCustomMob("minecraft:zombie", 0, 100, 0, {
        name: "§c§lСТРАЖ ПОДЗЕМЕЛЬЯ",
        health: 300.0,
        
        onTick: (mob) => {
            // 1. Ищем ближайшего игрока в радиусе 30 блоков
            var player = mob.findNearestPlayer(30);
            
            if (player != null) {
                var dist = mob.distanceToPlayer(player);
                
                // 2. Если игрок далеко, идем к нему
                if (dist > 3.0) {
                    mob.moveTo(player.getX(), player.getY(), player.getZ(), 0.4);
                } 
                // 3. Если игрок близко, атакуем его
                else {
                    mob.attackTarget(player);
                }
                
                // 4. Если здоровье падает ниже 30%, поджигаем себя и ускоряемся
                if (mob.getHealth() < 90.0) { // 30% от 300
                    mob.addEffect("minecraft:speed", 40, 1); // Скорость II на 2 секунды
                    mob.setOnFire(3);
                }
            } else {
                // Если игроков нет, останавливаемся
                mob.stopMoving();
            }
        },
        
        onDeath: (mob) => {
            // Проигрываем звук при смерти
            mob.playSound("minecraft:entity.wither.death", 1.0, 0.8);
            
            // Находим всех зомби в радиусе 10 блоков и убиваем их (взрыв при смерти)
            var minions = mob.findAllMobs("minecraft:zombie", 10);
            for (var i = 0; i < minions.length; i++) {
                minions[i].kill();
            }
            
            MCEngineServer.log("Страж повержен, его слуги уничтожены!");
        }
    });
});
```
